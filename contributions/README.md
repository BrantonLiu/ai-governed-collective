# 贡献记录目录

## 📋 说明

这个目录记录了所有成员的贡献。

## 📁 结构

```
contributions/
├── 2026-02/           # 2026年2月
│   ├── 2026-02-26.md
│   ├── 2026-02-27.md
│   └── ...
├── 2026-03/           # 2026年3月
│   └── ...
└── README.md          # 本文档
```

## 📝 如何记录贡献

1. 进入对应月份目录
2. 创建或打开当天的文件 `YYYY-MM-DD.md`
3. 按照 [模板](../templates/CONTRIBUTION-TEMPLATE.md) 填写贡献信息
4. 提交到Git仓库
