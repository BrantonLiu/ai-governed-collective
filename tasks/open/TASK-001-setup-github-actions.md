## 🎯 任务描述

设计并实现一个轻量级的贡献追踪系统，用于记录和评估成员的贡献。

### 核心功能
1. 贡献记录的增删改查
2. 贡献分的自动计算
3. 贡献报告的生成
4. 历史记录的查询

### 技术要求
- 使用Markdown作为数据存储格式
- 提供Python脚本用于数据解析和统计
- 支持Git作为版本控制
- 输出格式：JSON + Markdown报告

### 验收标准
- [ ] 能够解析贡献记录Markdown文件
- [ ] 能够计算个人和团队贡献分
- [ ] 能够生成周报和月报
- [ ] 提供使用文档和示例
- [ ] 代码提交到 `tools/` 目录
